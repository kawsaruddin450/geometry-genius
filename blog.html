1. What is DOM? what is the purpose of DOM?

~Document Object Model(DOM) is a programming interface for web documents. It represents the webpage as document, so that programs can have access to documents structure, style and content. DOM is mostly used for manipulating html document. Any html document can be manipulated by scripting languages such as javascript.
main purpose of DOM is improving web documents from static to dynamic. It can access, change, add or remove any property, element or content of document dynamically. That is the power of DOM and true purpose of using document object model.

3. What is Event bubble?

~Event bubbling is a method of event propagation in DOM when an event is in an element inside another element, and both elements have registered a handle to that event. It's a process that starts with the element that triggured the event and than bubbles up to the containing elements in the hierarchy. Event bubbling is generally started from the innermost element and then propagate through outer elements.

4. What is a callback function?

~A callback function is a function passed to another function as an argument. Then it is dependent on the caller function to do something.
